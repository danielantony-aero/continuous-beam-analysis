# Engineering Analysis Toolkit

This repository contains Python programs for basic structural analysis.

## Programs

### Continuous Beam Analysis
- Calculates reaction moments in a continuous beam system
- Supports:
  - Point load at center
  - Eccentric point load
  - Uniformly distributed load (UDL)
- Limitation:
  - Currently supports 2 spans only

### Beam Calculator
- Calculates bending moment, stress, moment of inertia and related values
- Supports different cross-sections
